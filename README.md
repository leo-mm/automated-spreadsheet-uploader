# automated-spreadsheet-uploader
Python automation that uploads and updates product prices on a website using Selenium and PyAutoGUI.

# Selenium File Upload Automation

This project automates the process of updating product prices on a website using Python.

## Technologies
- Python
- Selenium
- PyAutoGUI

## What the script does

1. Opens the website automatically
2. Scrolls the page to load all products
3. Clicks the "Load More" button
4. Scrolls back to the top
5. Uploads a spreadsheet file with updated prices
6. Updates the website automatically

## Goal

Automate a repetitive manual process and improve efficiency.
